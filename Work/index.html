MAXTube Internal Supports are compatible with the REV ION System and facilitate bolt-through assembly within a MAXTube without causing permanent deformation. Crucial for ensuring structurally sound through-bolted joints and allows multiple supports to be seamlessly slid into a MAXTube to accommodate multiple screws.

Features

  • Clearance for #10 screws
  • Provides screw placement in any available hole within MAXTube
  • Thin profile allows for multiple to be used on adjacent holes
